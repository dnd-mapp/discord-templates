# 🛡️ Server Rules & Roles

Welcome to the **D&D Mapp** hub. To ensure a productive environment for both players and developers, please adhere to the following guidelines.

### 📜 The Code of Conduct

1. **Be Respectful:** Treat all party members with kinship. Harassment, hate speech, and toxicity will result in an immediate Ban spell.
2. **Technical Discourse:** Keep high-level tech stack and architecture discussions in [#dev-general]. Keep gameplay and general chat in [#-general].
3. **No Spam / Self-Promo:** Please do not post unsolicited links or advertisements unless cleared by a Moderator.
4. **Issue Reporting:** Found a bug in the engine? Please use the proper GitHub issue template rather than tagging devs in general chat.
5. **AFK Policy:** Our **Inactive** voice channel is automated. If you are idle for more than 5 minutes in **General**, you will be moved to the Inactive channel to keep the lines clear.

### 🎭 Choose Your Role

React to this message to gain access to specific channels and notify the team of your interests:

🛠️ **Contributor** – *Access to development-heavy discussions and GitHub integration alerts.*  
🎲 **Dungeon Master** – *For those who run the games and want to discuss campaign management tools.*  
⚔️ **Player** – *For users focused on character sheets, VTT automation, and tactical combat.*  
🔔 **Update Ping** – *Get notified for new releases, patch notes, and deployment cycles.*

### 🗺️ Server Navigation

- **Need help?** Check [#onboarding] for a quick-start guide to the platform.
- **Latest Build:** Follow [#announcements] for new features and releases updates.
- **The Forge:** Dive into the code in [#dev-general].

> **Important**
