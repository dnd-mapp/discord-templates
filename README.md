# discord-templates

This repository serves as the central hub for channel structures and messaging templates used across the **dnd-mapp** Discord ecosystem. 

## 📦 What's Inside?

- **Channel Topics:** Standardized descriptions to keep channel purposes clear.
- **Onboarding & Welcome:** Templates for `#welcome`, `#rules-and-roles`, and `#onboarding` to streamline new member experiences.
- **Announcements:** Pre-formatted structures for updates, event launches, and maintenance alerts.
- **Resource Guides:** Initial messages for help or FAQ channels.

## 🚀 Usage

Each template is stored as a Markdown file. You can copy and paste the contents directly into your Discord channel settings or use them as a base for automated bot deployments.
